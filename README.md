# JetsonNano
JetsonNano를 사용하면서 필요 자료들이 난무하여 차후 활용이 편하도록 제작함.



## 개발환경
Model : JetsonNano Developer Kit 4GB (Memory Card O)

Volume : 256GB (Samsung Micro Memory Card)

Swap Memory : 8GB

JetPack Version : 4.6.0
